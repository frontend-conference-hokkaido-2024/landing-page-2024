## 開発環境構築
1. リポジトリをクローンする
`git clone https://github.com/frontend-conference-hokkaido-2024/landing-page-2024.git`

2. Node.jsのバージョンを20台にする

3. 依存関係をインストールする
`pnpm install`

4. 環境変数を設定する
`.env.local.sample`をコピーして`.env.local`を作成し、環境変数を設定する

5. 開発サーバーを起動する
`pnpm dev`
